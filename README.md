Project Overview
Weather App in Vanilla JavaScript - A web application designed to check the weather of any city using APIs from OpenWeatherMap.
Key Features
Pure JavaScript Implementation - Built entirely with vanilla JavaScript (no frameworks)
Live Weather Data - Fetches real-time weather information from openweathermap.org API
City Search - Allows users to search for weather by city name
Visual Weather Display - Shows temperature, weather conditions, "feels like" temperature, and min/max temperatures
Dynamic Backgrounds - Features visually appealing background images that change based on weather conditions
